# 🌸仿知乎日报项目🌸

基于React+Antd实现知乎日报浏览功能

## 🍄项目功能介绍：

本项目为移动端项目，采用REM响应式作为布局，主要功能分为五个功能模块：

- 首页浏览
  - 首页轮播图
  - 无限下拉
  - 按日期排序列表

- 登录/注册
  - 登录/注册验证
  - 验证码发送功能
  - 未注册用户自动注册
- 日报详情
  - 文章详情信息展示
  - 知乎评论跳转
- 用户个人中心
  - 用户信息修改
  - 退出登陆
- 文章收藏
  - 文章收藏列表
  - 文章收藏与取消收藏
